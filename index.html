  GoldChase Game

Maximum 5 players can play this game simultaneously from different machines to find real gold on a map from the fake golds. The first player to find real gold wins and the game ends. Players can communicate with each other by passing messages to individual player or broadcasting to everyone. Any player at any time can exit the game. The game is tested thoroughly for all the test cases. The entire game is developed using sockets, message passing queues and client-server model.
